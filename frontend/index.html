📚 StudyTask Manager

StudyTask Manager adalah aplikasi sederhana yang digunakan untuk mencatat dan mengelola tugas belajar agar lebih terorganisir.

Panduan Penggunaan

1. Jika sudah memiliki akun, klik tombol Login untuk masuk ke aplikasi.

2. Jika belum memiliki akun, klik tombol Register untuk membuat akun baru.

3. Setelah login, pengguna dapat menambahkan, mengedit, dan menghapus task pada halaman dashboard.
